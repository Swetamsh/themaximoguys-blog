## Research Protocol

### For Any Maximo Blog Post:

1. **Start with IBM Docs** (Tier 1)
   - Verify feature exists in target version
   - Get official terminology
   - Check for recent updates

2. **Cross-reference Community** (Tier 2)
   - Search Maximo Secrets for practical insights
   - Check Interloc for implementation guidance
   - Look for real-world experiences

3. **Find Code Examples** (Tier 3)
   - Check GitHub for samples
   - Verify code against current version
   - Test or validate before publishing

4. **Supplement with Video** (Tier 4)
   - IBM Mediacenter for official demos
   - YouTube for community tutorials

5. **Cite Everything**
   - Link to sources
   - Note version numbers
   - Timestamp for time-sensitive info
