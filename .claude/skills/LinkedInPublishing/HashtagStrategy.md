# Hashtag Strategy

## Placement Rules

- **Post body:** 3-5 hashtags at the very end of the post, on the last line
- **First comment:** NO hashtags — only blog link, @mentions, and engagement question
- **Never** put hashtags in the middle of post copy

## Selection (3-5 hashtags)

| Type | Count | Examples |
|------|-------|----------|
| Niche | 2 | `#IBMMaximo` `#MAS9` `#MaximoHealth` `#MaximoMobile` |
| Broad | 2 | `#AI` `#CloudMigration` `#DigitalTransformation` `#IoT` `#AssetManagement` |
| Brand | 1 | `#MaximoXcellence` |

## Post Flow

**Infographic → Hashtags → First Comment**

### Post Body (image + hashtags only)
```
#IBMMaximo #AI #CloudMigration #AssetManagement #MaximoXcellence
```

### First Comment
```
Full deep-dive: [blog URL]

@IBM @IBM Maximo

[Engagement question]
```
