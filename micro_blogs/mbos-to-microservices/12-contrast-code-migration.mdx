Before you migrate to MAS 9, every piece of custom code you own needs a color.

Not a guess. A deliberate assessment.

Use this traffic light system to categorize your inventory:

---

## GREEN — Low Risk. Migrate with standard testing.

- Simple automation scripts that use MboSet-based data access
- Standard MBO field defaults and attribute validation
- Basic business rules that do not reference IBM internal APIs
- App Designer layout customizations (screen definitions, field ordering)
- Integration scripts using standard Maximo REST endpoints

**What to do:** Plan migration. Test in non-production. Promote.

---

## YELLOW — Medium Risk. Evaluate before committing.

- Automation scripts that call external services (network connectivity review required)
- MBOs with moderate complexity and some IBM internal API dependencies
- App Designer customizations that include custom signatures or conditional logic
- Integration points with external systems that rely on specific Maximo behaviors
- Scripts that use patterns at the edge of what MAS Manage officially supports

**What to do:** Evaluate each item individually. Determine whether the logic can move to a script, external service, or whether MAS 9 handles it natively. Do not migrate blindly.

---

## RED — High Risk. Redesign conversation required.

- Any code using direct SQL queries against the Maximo database
- Calls to deprecated Maximo internal APIs that have changed or been removed
- Custom UI built on Dojo widgets, JSP templates, or custom Java beans for screen logic
- MBOs that hook deep into Maximo's class hierarchy or WebSphere-specific behavior
- Integrations that depend on WebSphere JMS queues, datasource JNDI names, or EJB patterns

**What to do:** Do not attempt to migrate these directly. Start a redesign conversation with your team and stakeholders now — before cutover planning begins. These take time.

---

Run every customization through this system before migration planning starts.

Your GREEN items give you confidence. Your YELLOW items give you a work plan. Your RED items give you the honest conversations you need to have early.
