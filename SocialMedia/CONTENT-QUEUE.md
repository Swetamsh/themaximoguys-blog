## Review Workflow

```
┌──────────┐    ┌───────────┐    ┌──────────┐    ┌───────────┐    ┌────────┐
│  DRAFT   │───→│ IN REVIEW │───→│ APPROVED │───→│ SCHEDULED │───→│ POSTED │
│          │    │           │    │          │    │           │    │        │
│ Created  │    │ Shared w/ │    │ All 4    │    │ Date+time │    │ Live   │
│ by Spdrm │    │ team for  │    │ approve  │    │ set in    │    │ URL    │
│          │    │ feedback   │    │          │    │ scheduler │    │ saved  │
└──────────┘    └───────────┘    └──────────┘    └───────────┘    └────────┘
       │              │                │
       │         Feedback?        Needs edits?
       │         Edit & re-review  Back to Draft
       └──────────────┘
```

### Roles
- **Creator:** Spiderman (AI) — generates posts + images
- **Reviewer 1:** You (User) — final approval authority
- **Reviewer 2-4:** 3 contributors — feedback + advocacy versions
- **Publisher:** Whoever has admin access to scheduling tool

### Review Checklist (for each post)
- [ ] Image looks good on mobile (not too small, text readable)
- [ ] Caption is accurate (no technical errors)
- [ ] Tone is right (professional but not corporate)
- [ ] Hashtags are correct (3-4, at the end)
- [ ] CTA is clear (link in comments, follow, etc.)
- [ ] Contributor advocacy version is personalized
- [ ] Scheduled date/time confirmed

## How to Use This File

1. I (Spiderman) create posts and add them here as 🟡 DRAFT
2. You review the image + caption (files linked above)
3. Share with contributors for feedback
4. Update status to 🟢 APPROVED when ready
5. Schedule in Notion or scheduling tool → update to 📅 SCHEDULED
6. After posting, add the live URL → update to ✅ POSTED

This file is the single source of truth for what's queued, what's approved, and what's live.
