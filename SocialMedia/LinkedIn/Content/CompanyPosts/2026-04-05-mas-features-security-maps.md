## Post Copy

```
Three tickets on Monday morning. Three different subsystems. One root cause.

MAS 9 fundamentally rewrote authentication, API security, and mapping.

-> Google Maps and Bing Maps are deprecated -- OpenMap is the new default, ArcGIS for enterprise GIS
-> Authentication now requires an external IdP -- SAML, OIDC, LDAP, or MongoDB local
-> Suite-level SSO is built-in -- no more manual SSO configuration per application
-> API Keys replace basic auth for integrations -- with scope control and automatic rotation
-> MAS 9.1 adds multiple IdP fallback, self-registration, and multiple LDAP server support

If your migration planning has not accounted for these changes, this is your wake-up call.

Full article in comments. Save this for your team.
```
