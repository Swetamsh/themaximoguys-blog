## Post Copy

```
Four dedicated mobile supply chain apps replace everything Maximo Anywhere offered -- and then some.

-> Issues and Transfers Mobile: staging, barcode scanning, storeroom-to-storeroom transfers, tools transfer, full offline
-> Inventory Count Mobile: barcode-driven counting, blind count mode, photo attachments, approval workflows -- but reconciliation is online only
-> Receiving Mobile: PO-based receipts, rotating asset receipt, inspection on receipt -- MAS 9.1 adds bin updates and return processing
-> Technician App: field material requests, usage recording, barcode scanning, planned material views, availability checks
-> Migration from Anywhere eliminates device-side builds entirely -- server-side configuration only

No more MAF rebuilds. No more manual APK distribution. No more XML customization. Server-side config, instant updates, every device.
```
