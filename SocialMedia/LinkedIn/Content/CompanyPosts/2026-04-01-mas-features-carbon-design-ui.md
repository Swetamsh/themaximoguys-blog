## Post Copy

```
Your users will not recognize Maximo when they log into MAS 9.

This is not a skin change. IBM replaced the entire UI framework with Carbon Design System.

-> All legacy skins are gone -- no tivoli09, no tivoli13, no choices
-> Field labels are permanently above fields -- the vertical toggle is gone
-> CSS class names changed from .bx-- to .cds-- between MAS 9.0 and 9.1
-> Custom CSS is unsupported by IBM -- if it breaks, you are on your own
-> Every screenshot in your SOPs and training materials is now wrong

The user training gap is massive. Budget for comprehensive retraining, not a 2-hour overview. Teams report 60-70% productivity in week one, with 3-4 months to full comfort.

This is not optional preparation. This is the cost of the upgrade.

#IBMMaximo #CarbonDesign #MAS9 #TheMaximoGuys #UX
```
