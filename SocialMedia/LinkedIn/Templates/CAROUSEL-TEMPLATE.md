# LinkedIn Carousel Post Template

## Format Rules
- **Slides:** 8-12 slides (sweet spot: 10)
- **Dimensions:** 1080 x 1350px (portrait, 4:5 ratio)
- **File type:** PDF upload (LinkedIn native document)
- **Font size:** Minimum 24pt for readability on mobile
- **Brand colors:** Use TheMaximoGuys brand palette consistently

## Slide Structure

### Slide 1: Hook (Title Slide)
```
[Bold provocative question or statement]

Example:
"7 Things That Changed Forever
When Maximo Became MAS"

TheMaximoGuys
```
- Must stop the scroll
- No more than 8 words
- Include brand logo/name

### Slide 2: Context/Problem
```
[Set the scene — what pain point are we addressing?]

Example:
"If you're still running Maximo 7.6,
you're managing an asset platform
that IBM considers legacy.

Here's what changed — and why it matters."
```

### Slides 3-9: Content Slides (one idea per slide)
```
[Number]. [Key Point Headline]

[2-3 sentences of explanation]

[Visual: diagram, icon, or screenshot if applicable]
```
- ONE concept per slide
- Use numbered lists for progression
- Include data points where possible
- Mix text slides with visual/diagram slides

### Slide 10: Summary/Recap
```
Quick recap:
1. [Point 1]
2. [Point 2]
3. [Point 3]
...

Save this for reference.
```

### Slide 11: CTA (Final Slide)
```
Want the full deep-dive?

Read our complete [Series Name] guide
→ Link in comments

Follow TheMaximoGuys for weekly
Maximo technical content.

#IBMMaximo #EAM #TheMaximoGuys
```

## Caption Template (for the post itself)
```
[Hook question — 1 line that makes them stop scrolling]
```
