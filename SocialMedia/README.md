## Content Pipeline

```
Blog Post (77 existing across 8 series)
    │
    ├─→ LinkedIn: Carousel + Text Post + Poll + Newsletter
    ├─→ Instagram: Carousel (repost) + Reel + Story
    ├─→ YouTube: Tutorial + Short (cross-post from Reel)
    ├─→ X/Twitter: Thread + Single tweets
    ├─→ Email: Newsletter edition section
    ├─→ Medium/dev.to: Syndicated article (canonical URL)
    ├─→ Reddit: Organic answers using blog knowledge
    ├─→ IBM TechXchange: Community contributions
    ├─→ GitHub: Code samples from technical posts
    └─→ Podcast: Audio version of blog post
```
